Mapping Berlin

provides my very own map of Berlin.

Genesis

  • Made from own data – initially aggregated during my participation on OpenStreetMap from 2009 to 2012 – and supplemented by public domain Landsat imagery
  • Using the OSM toolchain: BT747, gpgbabel, custom Perl scripts, JOSM, Maperitive, Leaflet & OpenLayers
  • Still extended and updated occasionally
  • Originally inspired by the need of an attribution-free and free background map for another geospatial datamining project

See it in action

  • No distractions, no street names
  • providing the pure map experience
  • Two flavours:
  • Lightweight, mobile-friendly using Leaflet
  • As Overlay on an OSM-Mapnik map using OpenLayers
